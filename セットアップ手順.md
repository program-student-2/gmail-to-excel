# 就活メール → Excel 自動転記システム

Gmailに届く就活マイページの登録完了メールから、**企業名・ID・パスワード・ログインURL**を
自動で読み取り、`サマーインターン管理.xlsx` に転記するシステムです。

- メール取得：**Gmail API**（自動取得）
- 情報抽出：**Claude API**（書式がバラバラでもAIが読み取る）
- 実行：**run.bat をダブルクリック**

---

## 必要な準備（初回のみ・3ステップ）

### ステップ1：Anthropic（Claude）APIキーを用意

1. https://console.anthropic.com/ にログイン
2. 「API Keys」→「Create Key」でキーを作成（`sk-ant-...`）
3. このフォルダの `.env.example` を **コピーして `.env` にリネーム**
4. `.env` をメモ帳で開き、キーを貼り付けて保存：
   ```
   ANTHROPIC_API_KEY=sk-ant-ここに自分のキー
   ```

> 料金：1通あたり数円程度です。コストを抑えたい場合は `config.json` の
> `"model"` を `"claude-haiku-4-5"` に変えると安くなります（精度は少し下がります）。

### ステップ2：Gmail API の認証ファイル（credentials.json）を用意

1. https://console.cloud.google.com/ にアクセス（Googleアカウントでログイン）
2. 上部のプロジェクト選択 →「新しいプロジェクト」を作成（名前は任意）
3. 左メニュー「APIとサービス」→「ライブラリ」→ **「Gmail API」を検索して有効化**
4. 「APIとサービス」→「OAuth同意画面」を設定
   - User Type：**外部**
   - アプリ名・メールアドレスなど必須項目を入力
   - 「対象ユーザー（テストユーザー）」に **自分のGmailアドレスを追加**
5. 「APIとサービス」→「認証情報」→「認証情報を作成」→ **「OAuth クライアント ID」**
   - アプリの種類：**デスクトップアプリ**
6. 作成後に表示される「JSONをダウンロード」を押し、ファイル名を **`credentials.json`** に変更して、
   **このフォルダ（`run.bat` と同じ場所）に置く**

### ステップ3：設定ファイル（config.json）を用意

1. `config.example.json` を **コピーして `config.json`** にリネーム
2. メモ帳で開き、`excel_path` を自分のExcelのフルパスに書き換える
   （`\` は2つ重ねて書く。例：`C:\\Users\\名前\\Downloads\\サマーインターン管理.xlsx`）

### ステップ4：Python を用意（未インストールの場合）

- https://www.python.org/downloads/ から Python 3.10以上 をインストール
- インストール時に **「Add Python to PATH」にチェック** を入れてください

---

## 使い方（毎回）

1. **転記先のExcel（サマーインターン管理.xlsx）を閉じておく**（開いていると保存できません）
2. **`run.bat` をダブルクリック**
3. 初回だけブラウザが開き、Gmailへのアクセス許可を求められます → 自分のアカウントで許可
   - 「このアプリは確認されていません」と出たら「詳細」→「（アプリ名）に移動」で進めます
4. 解析が始まり、終わると Excel に自動で書き込まれます

実行すると、こんな表示が出ます：
```
[2/4] メールを解析して Excel に転記中...
   1. [新規] 野村総合研究所 … ID=28015808, URL=..., マイページ登録=済
   2. [更新] トヨタ … URL=...
   3. [対象外] 【セール】お得な情報
...
[4/4] 完了
      新規追加: 2 件 / 既存更新: 1 件
```

## 動作の仕組み（安心ポイント）

- **既存データは上書きしません**（空欄のセルだけ補完）。
  既存の値も更新したい場合は `config.json` の `"overwrite_existing"` を `true` に。
- **企業名の表記ゆれに対応**（例：野村證券＝野村証券、トヨタ＝トヨタ自動車）。
  同じ企業は既存の行に統合し、重複行を作りません。
- 処理したメールには Gmail に **「就活処理済み」ラベル** を付け、次回は二重処理しません。
- マイページ登録以外のメール（広告など）は **[対象外]** として転記しません。

---

## 設定（config.json）

| 項目 | 説明 |
|------|------|
| `excel_path` | 転記先Excelのフルパス |
| `model` | 使うClaudeモデル（`claude-opus-4-8` / 安価な `claude-haiku-4-5`） |
| `gmail_query` | 取得するメールの検索条件（Gmailの検索式が使えます） |
| `max_emails` | 1回で処理する最大件数 |
| `mark_processed` | 処理済みラベルを付けるか（`true`/`false`） |
| `overwrite_existing` | 既存の値も上書きするか |
| `mypage_mark` | マイページ登録欄に入れる文字（既定は「済」） |

---

## 困ったとき

| 症状 | 対処 |
|------|------|
| `ANTHROPIC_API_KEY が設定されていません` | `.env` を作成し、キーを記入したか確認 |
| `credentials.json が見つかりません` | ステップ2のファイルをこのフォルダに置く |
| `Excelファイルが開いたままです` | Excelを閉じてから再実行 |
| メールが0件になる | 既に「就活処理済み」ラベルが付いている可能性。Gmailでラベルを外すか、`gmail_query`を調整 |
| 認証をやり直したい | `token.json` を削除してもう一度実行 |

| `extractor.py` | Claudeで情報抽出 |

| `credentials.json` | Gmail認証（自分で配置・**共有禁止**） |
